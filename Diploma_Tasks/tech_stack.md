# Задача №6 - выбрать технологический стек для разработки

## Процесс решения

Для разработки программной части проекта был выбран язык программирования С++ и фреймворк Qt.
Выбор данного языка программирования обоснован б**о**льшим контролем над процессами внутри исполняемого кода. Из преимуществ можно выделить следующие пункты:

- фиксированное выделение памяти под рабочие задачи
- быстрота работы
- тип языка, а именно компилируемый
- поддержка принципов ООП

Выбор фреймворка обоснован возможностями создания кросс-платформенных проектов. Планируемый веб-сервис создаётся для заполнения документов РКФ через мобильные устройства. Проектирование изначального интерфейса и функционала для персонального компьютера или ноутбука не ставит ограничений по его портированию на другие платформы.
